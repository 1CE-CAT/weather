Возможности
  Ввод географических координат (широта и долгота)
  Отображение текущих погодных условий
  Показ температуры в градусах Цельсия
  Отображение скорости и направления ветра
  Интерактивная карта местности через OpenStreetMap
  Визуальное представление погодных условий через иконки
  Возможность добавления нескольких виджетов погоды
  Удаление отдельных виджетов
  В зависимости от погоды изменяется ее иконка

Технические особенности
  Валидация данных
  Проверка корректности введенных координат
  Валидация числовых значений широты и долготы
  Проверка диапазонов: широта от -90 до 90, долгота от -180 до 180

API интеграция
Приложение использует Open-Meteo API для получения актуальных данных о погоде.
